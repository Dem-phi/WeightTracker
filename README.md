# WeightTracker iOS 应用 - 指南

## 快速开始

WeightTracker 是一个使用 SwiftUI 和 CoreData 构建的双人体重追踪应用。本指南帮助你快速设置、测试和迭代。

## 项目设置

```bash
# 克隆并打开项目
open WeightTracker.xcodeproj
```

**系统要求：**
- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+
